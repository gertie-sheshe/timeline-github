# GitHub Timeline App
See the timeline of a user's GitHub repositories.

## Running Locally
1. Clone the repo.

2. Install using `npm i`

3. Run using `yarn start` and visit app on localhost:3000

## Technologies
React. Redux. Redux Thunk. Reselect. React Testing Library.
